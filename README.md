# Week 1 Homework

In this repository, Adam Czopp, Antony Siahaan and myself (Linus Kelsey) have produced our homework for the first week of the Encode Club's Q2 2023 2PM Solidity Bootcamp.

The contract we used is found in `HelloWorld.sol`, and `artifacts` contains the related `.json` files.
